## What I'm working on

### 🔐 [slayer-vault](https://github.com/cardoza1991/slayer-vault) — *Go*
Self-hosted AI credential isolation platform for secure, secretless agent access. Lets autonomous agents call third-party APIs without ever seeing the raw secrets.

### 🔁 [Driftless](https://github.com/cardoza1991/Driftless) — *Go / Shell*
AI-powered sync engine that keeps your frontend and backend in lockstep — automatically. Catches drift between contracts and implementations before it ships.

### 🤖 [droidz](https://github.com/cardoza1991/droidz) — *Go*
Lightweight agent runtime experiments — building blocks for the tooling above.

### 🎙️ [Sunshine Microphone Pass-Through](https://github.com/cardoza1991/sunshine-mic-passthrough) — *C++*
Bidirectional mic support for the [Sunshine](https://github.com/LizardByte/Sunshine) game streaming server. Cross-platform implementation (Linux/Windows/macOS) addressing a long-standing community request — [upstream PR #4078](https://github.com/LizardByte/Sunshine/pull/4078).
